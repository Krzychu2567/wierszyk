# wierszyk
Abecadło z komina spadło,
O ziemię się wywalić,
Rozsypało się po rogach,
Strasznie się rąbnęło:
I - zgubiło kropeczkę,
H - złamało kładeczkę,
B - zbiło sobie brzuszki,
A - zwichnęło nóżki,
O - jak balon pękło,
aż się P przelękło,
T - daszek zgubiło,
L - do U wskoczyło,
S - się wyprostowało,
R - prawą nogę złamało,
W - stanęło do góry dnem i udaje, że jest M.
